// 1. Headings

- Item 1
- Item 2
  - Nested Item

// 2b. Ordered Lists
1. First

//Use `backticks` for inline code.

// Links & Images

// 6. Quotes & Callouts
> This is a blockquote.
> It’s good for notes or emphasis.

| Name | Age | Job |
|------|-----|-----|
| Riley | 21 | Student |
| Alex  | 22 | Developer |

---

- [ ] To do  
- [x] Done

That is so funny! :joy:

I need to highlight these ==very important words==.

H~2~O
